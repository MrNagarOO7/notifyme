# NotifyMe Backend
It's server part of NotifyMe Application.
## Description
We can notify ourself for event and any reminder.
## Getting Started

### Dependencies

* NodeJs

### Executing program

* Install npm dependencis
```
npm install
```
* Run Script to start
```
npm start
```
* Run Monitor Script
```
npx nodemon app.js
```
